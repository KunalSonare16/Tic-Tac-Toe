# 🎮 Tic Tac Toe Game

A simple and interactive **Tic Tac Toe web application** built using **React (Vite), HTML, and CSS**. This project allows two players to play the classic game with a clean UI and smooth gameplay experience.

---

## 🌐 Live Demo

👉 https://tic-tac-toe-theta-coral.vercel.app/

---

## 🚀 Features

* 🧑‍🤝‍🧑 Two-player gameplay (X vs O)
* 🎯 Click-based board interaction
* 🏆 Automatic winner detection
* 📜 Move history tracking (Log system)
* 🔄 Restart / Game Over handling
* ⚡ Fast performance using Vite

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🧱 HTML5
* 🎨 CSS3
* 🚀 Vercel (Deployment)

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/tic-tac-toe.git
```

2. Navigate into the project:

```bash
cd tic-tac-toe
```

3. Install dependencies:

```bash
npm install
```

4. Run the development server:

```bash
npm run dev
```
